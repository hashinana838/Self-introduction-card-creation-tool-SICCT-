# 自己紹介カードを生成するツール
ハッカソンで作成したwebアプリ
## 目標
- 自己紹介カードをwebで作って、画像にして、ツイートする
- 全員がgitを使えるようになる
- 最終的には色んなAPIを駆使して、ユーザー名やIDを入れるだけで良い感じに生成してくれるようにしたい
